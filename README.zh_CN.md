本项目是从Ubuntu 发行版本中的ZooKeeper 原生C语言函数库导入而来。⚠️本项目只能在Ubuntu 16.04上编译⚠️

本项目是[Perfect](https://github.com/PerfectlySoft/Perfect) 项目的组成部分，本项目所有文件都是为支持SPM软件包管理器开发而设置的，无需手工下载或编译。

## 编译选项

为了增加对多线程的支持，您可能需要使用以下编译标识符号：

```
$ swift build -Xlinker -dthreaded
$ swift test -Xlinker -dthreaded
```
